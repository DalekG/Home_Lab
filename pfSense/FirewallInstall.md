# Install

## Download
- Navigate to (**pfSense**)[https://www.pfsense.org/download/]
- Click on "Download"
- Select the image type
    - Preferably memstick option
- Add to cart
- Enter cart
- Checkout
- Enter your information and complete checkout
- Let download finish

## BalenaEtcher
- Navigate to (**belenaEtcher**)[https://etcher.balena.io/]
- Click "Download Etcher"
- Select the correct download for you OS
- Let the download finish

## Prepare the USB
- Insert your USB
- Open balenaEtcher
- Select "Flash from file"
- Select the netgate-installer-{image}.img.gz
- Select target
    - Location of your USB
- Flash!
